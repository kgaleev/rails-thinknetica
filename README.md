# README

Technology Stack:
* Ruby 3.2.2
* Rails 6.1.7.3


Entities in the project:
* Admin 
* User 
* Test
* Question
* Answer
* Result


Possible Actions:
* Admin AND User have Authorization
* Admin can create and manage tests: add questions and answers
* User can take tests
* Admin AND User can view results
* User can publish results in social networks
